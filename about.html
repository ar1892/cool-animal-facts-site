About This Site

Hi! I'm Andres Raygada, and I created this website as part of a class project to explore the unique and incredible biodiversity of Peru. As someone who loves both technology and nature, I wanted to showcase some of the amazing animals that call Peru home.

This site features six fascinating species — from the elusive Spectacled Bear to the colorful Andean Cock-of-the-Rock. The goal is to raise awareness and appreciation for Peru’s wildlife, and to give visitors a simple, engaging way to learn something new.

The site was built using HTML, CSS, and the Bootstrap framework for responsiveness. It is hosted publicly on GitHub Pages.

Thank you for visiting!
